# Demo Script

"ArbiX no intenta predecir Bitcoin. Lo que hace es detectar ineficiencias reales entre exchanges. El sistema consume order books en tiempo real por WebSocket, normaliza los datos, compara el mejor ask contra el mejor bid de otros exchanges, calcula rentabilidad neta despues de fees, slippage y latencia, y solo simula la operacion si supera el umbral de riesgo. Ademas, maneja liquidez parcial, actualiza wallets virtuales y registra el P&L acumulado."

## Orden De Presentacion

1. Mostrar dashboard principal.
2. Explicar exchanges conectados.
3. Mostrar BTC/USDT y ETH/USDT en Market Matrix.
4. Mostrar mejor bid/ask resaltado por par.
5. Mostrar oportunidad detectada.
6. Abrir detalle de oportunidad.
7. Explicar calculo bruto vs neto.
8. Mostrar fees, slippage, VWAP y latencia end-to-end.
9. Mostrar si se ejecuto o rechazo y por que.
10. Abrir Execution Simulator.
11. Mostrar timeline, order book depth y wallet deltas.
12. Mostrar Wallets actualizadas.
13. Mostrar P&L acumulado.
14. Mostrar Risk Center y circuit breaker.
15. Activar un escenario desde Demo Control Panel.
16. Mostrar Strategy Lab triangular watch-only.
17. Cerrar con arquitectura, README y decisiones tecnicas.
